## 🧠 What is Sentira CORE?

<div align="center">

> *"Language carries emotion. Sentira CORE makes the invisible visible."*

</div>

**Sentira CORE** is a next-generation **Neural Sentiment Engine** — a production-grade, interactive Streamlit application powered by advanced NLP and machine learning. It analyzes raw text with surgical precision to detect and classify complex human emotions in real time.

Whether you're a **developer** building empathetic AI products, a **researcher** studying linguistic patterns, or an **analyst** extracting emotional signals from data — Sentira CORE delivers **high-fidelity emotional intelligence** wrapped in a stunning glassmorphism UI.

## 🚀 Key Features

<table>
<tr>
<td width="50%">

### 🔮 Neural Sentiment Processing
Leverages a **Linear Support Vector Classifier (LinearSVC)** trained on thousands of labeled emotional text samples to detect and decode emotional tone with exceptional precision.

</td>
<td width="50%">

### 🌈 Emotional Spectrum Index
Classifies text across **6 emotional dimensions** — Joy, Love, Surprise, Sadness, Anger, and Fear — each paired with a **Neural Resonance Score** (confidence %).

</td>
</tr>
<tr>
<td width="50%">

### 🎨 Ultra-Modern Glassmorphism UI
Translucent containers, background blur effects, neon accent animations, and pulsing UI elements — all built inside Streamlit for a native-app feel.

</td>
<td width="50%">

### ⚡ Real-Time Softmax Calibration
Decision function outputs are converted through a **softmax layer** to produce human-readable confidence probabilities for every emotional class simultaneously.

</td>
</tr>
</table>

## ⚙️ How It Works

```
┌─────────────────────────────────────────────────────────────┐
│                  SENTIRA CORE — PIPELINE                    │
└─────────────────────────────────────────────────────────────┘

  📝 RAW TEXT INPUT
        │
        ▼
  🔧 PREPROCESSING
   └─ Lowercasing · Punctuation Removal · ASCII Filter · NLTK Stopwords
        │
        ▼
  📊 TF-IDF VECTORIZATION
   └─ Sparse matrix representation of token frequency × inverse doc frequency
        │
        ▼
  🧠 LINEAR SVC MODEL
   └─ Hyperplane decision boundary across 6 emotion classes
        │
        ▼
  🌡️ SOFTMAX CALIBRATION
   └─ Raw decision scores → Normalized confidence percentages
        │
        ▼
  💠 VISUAL OUTPUT
   └─ Primary emotion card + confidence score + full spectrum breakdown
```

## 📦 Local Setup

```bash
# 1️⃣ Clone the repository
git clone <your-repo-url>
cd sentira-core

# 2️⃣ Create & activate virtual environment
python -m venv venv

# Windows
.\venv\Scripts\activate
# macOS/Linux
source venv/bin/activate

# 3️⃣ Install dependencies
pip install streamlit joblib nltk scikit-learn scipy pandas

# 4️⃣ Launch 🚀
streamlit run app.py
```

> **Tip:** The app will auto-open at `http://localhost:8501` — no browser config needed.
